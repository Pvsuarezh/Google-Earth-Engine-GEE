# Análisis Geoespacial para Evaluación de Riesgos y Prevención de Desastres con Google Earth Engine

Este repositorio recopila scripts y recursos utilizados en proyectos de evaluación de riesgos, monitoreo de eventos climáticos extremos, prevención y mitigación de desastres naturales mediante Google Earth Engine.

## Descripción del Proyecto

El objetivo es llevar a cabo un análisis detallado de riesgos asociados a desastres naturales, incluyendo monitoreo de eventos climáticos extremos, evaluación de riesgos, y estrategias de prevención y mitigación.

## Contenido

- **Scripts:** Scripts de GEE para el análisis de riesgos, detección de eventos climáticos extremos, y evaluación de áreas propensas a desastres.
- **Recursos:** Conjuntos de datos relevantes como imágenes satelitales, datos meteorológicos, y mapas de riesgo.

## Ejecución

Los scripts están diseñados para ser ejecutados en la plataforma Google Earth Engine y están documentados para su fácil uso y comprensión.

## Fuentes de Datos

Se utilizan datos provenientes de fuentes satelitales, estaciones meteorológicas, y modelos climáticos disponibles en Google Earth Engine.
